# **App Name**: Fibonacci Dice Roller

## Core Features:

- Dice Roll: Button to trigger a virtual dice roll that randomly selects a Fibonacci number.
- Animated Dice: Display an animated graphical representation of the dice rolling.
- Fibonacci Number Display: Clear display of the rolled Fibonacci number or symbol (e.g., 1, 2, 3, 5, 8, 13, 21, '?', 'â˜•').

## Style Guidelines:

- Primary color: Clean, professional purple(#a61f7d), inspired by clarity and reliability, providing a focused estimation environment.
- Background color: Soft white (#f9f9f9), close in hue to the primary, maintaining a light and uncluttered backdrop.
- Accent color: Light blue (#323e97), analogous to the primary, to highlight key interactive elements with clear visual distinction.
- Body and headline font: 'Inter', a grotesque-style sans-serif, suitable for a modern and objective look.
- Use clear, geometric icons for the dice and interface elements.
- Smooth, subtle animations for the dice roll and number display.
- Clean, modern layout with a focus on readability and ease of use.
